# Condecora

Sistema para gerenciamento de especialidades, controle de estoque e acompanhamento.

## Funcionalidades

- Cadastro de especialidades
- Controle de estoque atual
- Controle de quantidade necessária
- Cálculo de itens faltantes
- Gerenciamento e acompanhamento de condecorações
